# Piper OpenVINO Home Assistant App

Custom Home Assistant app for Piper text-to-speech using ONNX Runtime OpenVINO
acceleration on Intel GPUs.

## App

- Name: Piper OpenVINO
- Slug: `piper_openvino`
- Image: `ghcr.io/wheemer/amd64-app-piper-openvino`
- Supported architecture: `amd64`
